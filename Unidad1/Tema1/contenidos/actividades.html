Actividad · Aprendizaje Autónomo

Ejemplo práctico de Aprendizaje Autónomo

Caso realista para analizar fases, estrategias y evidencias de aprendizaje autorregulado.

Datos del caso

Nombre del estudiante
Laura Pérez
Meta de aprendizaje
Aprender los fundamentos de HTML y CSS para diseñar una página web básica.
Motivación
Aplicar a una pasantía donde se exige conocer diseño web.
Duración del plan
4 semanas

Acciones autónomas realizadas

  • Estableció un plan de estudio de 4 semanas.
  • Utilizó la plataforma freeCodeCamp.org para practicar con ejercicios interactivos.
  • Tomó notas y creó un glosario con los conceptos nuevos.
  • Diseñó una página web personal como proyecto final.
  • Evaluó su avance haciendo una autoevaluación semanal.
  • Buscó retroalimentación de sus compañeros a través de un foro.

Sugerencia: agrega enlaces a evidencias (repositorio, capturas, documentos de notas).

Seguimiento y autoevaluación (semanal)

Semana Objetivo Autoevaluación (1–5) Notas / Ajustes
1 Estructura básica de HTML (etiquetas, jerarquía, semántica). 4 Repasar etiquetas semánticas <header>, <main>, <footer>.
2 Selectores y cascada en CSS, box model. 3 Practicar más ejercicios de especificidad.
3 Layout con Flexbox y Grid. 4 Crear mini-layouts y clonar un diseño simple.
4 Proyecto personal y publicación. 5 Pulir accesibilidad básica y responsive.

Resultado

Laura logró crear una página web funcional, entendió la estructura del código HTML y los estilos CSS, y pudo mostrar su trabajo en una entrevista.

Actividades para el aula / taller

  1. Identifica la fase de planificación, ejecución y autorreflexión en el caso.
  2. Redacta dos indicadores de logro (SMART) para la semana 2.
  3. Propón dos mejoras a la estrategia de retroalimentación utilizada.
  4. Enlista tres evidencias verificables del aprendizaje logrado.
